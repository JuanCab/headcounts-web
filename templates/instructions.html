Course Headcounts Instructions

All "navigation" is done by editing the URL in the address bar of your browser.

The first thing after the site name is one of the items below. If that is the only thing you specify you will see data for the current or upcoming semester (depending on time of semester)

  • a subject (e.g. phys or econ, upper or lower case is fine) -- show courses in that subject
  • a college code (allowed codes are 'cbac', 'coah', 'cshe', 'cehs', or 'none')
  • all -- show all courses.
  • lasc -- shows all lasc courses.
  • wi -- show all writing intensive courses.
  • 18online -- show all 18-Online courses.

You can narrow the results by adding any of these as the next part of the URL; add more than one to get more specific:

  • a "year term" code -- show all courses for specific year/term
  • a course number -- show that course for all terms (if the course number ends with an underscore, it will match all courses starting with the digits before the underscore)
  • an LASC area -- show only courses in that area for all years
  • all -- show all terms.
  • 18online -- show all 18-Online courses.

Examples (NOTE: In general, matching is case-insensitive):

  • Current/upcoming Math courses: {{ base_url }}math
  • Current/upcoming courses in College of Business, Analytics, and Communication: {{ base_url }}cbac
  • Current/upcoming writing-intensive courses: {{ base_url }}wi
  • Math 261: {{ base_url }}math/261
  • Math 261, Fall 2014 only: {{ base_url }}math/261/20153
  • CSIS 349L, Fall 2024 only: {{ base_url }}csis/349l/20253
  • MUS 156 (any variant), Fall 2024 only: {{ base_url }}mus/156_/20253 (an underscore '_' matches any variant of MUS 156)
  • Art courses, Fall 2014: {{ base_url }}art/20153
  • LASC area 2 courses, Spring 2016: {{ base_url }}lasc/2/20165
  • Everything...all courses, all years: {{ base_url }}all
