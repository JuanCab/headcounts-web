Instructions

All "navigation" is done by editing the URL in the address bar of your browser.

The first thing after the site name is one of the items below. If that is the only thing you specify you will see data for the current or upcoming semester (depending on time of semester)

  • a subject (e.g. phys or econ, upper or lower case is fine) -- show courses in that subject
  • all -- show all courses.
  • lasc -- shows all lasc courses.
  • wi -- show all writing intensive courses.
  • 18online -- show all 18-Online courses.

You can narrow the results by adding any of these as the next part of the URL; add more than one to get more specific:

  • a "year term" code -- show all courses for specific year/term
  • a course number -- show that course for all terms
  • an LASC area -- show only courses in that area for all years
  • all -- show all terms.
  • 18online -- show all 18-Online courses.

Examples:

  • Current/upcoming Math courses: http://headcounts.herokuapp.com/math
  • Current/upcoming writing-intensive courses: http://headcounts.herokuapp.com/wi
  • Math 261: http://headcounts.herokuapp.com/math/261
  • Math 261, Fall 2014 only: http://headcounts.herokuapp.com/math/261/20153
  • Art courses, Fall 2014: http://headcounts.herokuapp.com/art/20153
  • LASC area 2 courses, Spring 2016: http://headcounts.herokuapp.com/lasc/2/20165
  • Everything: http://headcounts.herokuapp.com/all

Disclaimer/Data Source

These are NOT real time results. The approximate time that the data was generated is shown above each table. All data is scraped from the public-facing MinnState-maintained course search tool.
